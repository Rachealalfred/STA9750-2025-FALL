Executive Summary

This report analyzes Netflix’s global viewing data to highlight the streaming giant’s most successful programming. Based on publicly available Top 10 data, we examine viewing patterns across countries, languages, and content types to prepare press releases celebrating Netflix’s achievements.

Data Acquisition and Preparation

Download Data Files

{r setup, warning=FALSE, message=FALSE} # Create directory for data if(!dir.exists(file.path(“data”, “mp01”))){ dir.create(file.path(“data”, “mp01”), showWarnings=FALSE, recursive=TRUE) }

Download Global Top 10 data

GLOBAL_TOP_10_FILENAME <- file.path(“data”, “mp01”, “global_top10_alltime.csv”) if(!file.exists(GLOBAL_TOP_10_FILENAME)){ download.file(“https://www.netflix.com/tudum/top10/data/all-weeks-global.tsv”, destfile=GLOBAL_TOP_10_FILENAME) }

Download Country Top 10 data

COUNTRY_TOP_10_FILENAME <- file.path(“data”, “mp01”, “country_top10_alltime.csv”) if(!file.exists(COUNTRY_TOP_10_FILENAME)){ download.file(“https://www.netflix.com/tudum/top10/data/all-weeks-countries.tsv”, destfile=COUNTRY_TOP_10_FILENAME) }

Load Required Libraries

{r libraries, warning=FALSE, message=FALSE} library(readr) library(dplyr) library(stringr) library(DT) library(lubridate)

Import and Clean Data

Task 01: Fix Global Data NA Values

{r import-global, warning=FALSE, message=FALSE} # Read global data GLOBAL_TOP_10 <- read_tsv(GLOBAL_TOP_10_FILENAME, show_col_types = FALSE)

Convert “N/A” strings to proper NA values

GLOBAL_TOP_10 <- GLOBAL_TOP_10 %>% mutate(season_title = ifelse(season_title == “N/A”, NA, season_title))

Verify the fix

glimpse(GLOBAL_TOP_10)

Task 02: Import Country Data with NA Handling

{r import-country, warning=FALSE, message=FALSE} # Read country data with NA values handled at import COUNTRY_TOP_10 <- read_tsv(COUNTRY_TOP_10_FILENAME, na = “N/A”, show_col_types = FALSE)

Verify the structure

glimpse(COUNTRY_TOP_10)

Exploratory Data Analysis

Sample of Global Top 10 Data

{r display-sample} set.seed(123)

GLOBAL_TOP_10 %>% slice_sample(n = 10) %>% mutate(runtime_minutes = round(60 * runtime)) %>% select(week, category, weekly_rank, show_title, weekly_hours_viewed, weekly_views, runtime_minutes) %>% rename( Week = week, Category = category, Rank = weekly_rank, Show Title = show_title, Hours Viewed = weekly_hours_viewed, Views = weekly_views, Runtime (min) = runtime_minutes ) %>% datatable( options = list(pageLength = 10), rownames = FALSE ) %>% formatCurrency(c(‘Hours Viewed’, ‘Views’), currency = ““, interval = 3, mark =”,“, digits = 0)

Task 04: Top Non-English Film by Weeks in Top 10

{r top-non-english-film} top_non_english <- GLOBAL_TOP_10 %>% filter(category == “Films (Non-English)”) %>% group_by(show_title) %>% summarise(total_weeks = sum(cumulative_weeks_in_top_10, na.rm = TRUE)) %>% ungroup() %>% arrange(desc(total_weeks)) %>% slice(1)

The non-English film with the most cumulative weeks in the global top 10 is *r top_non_english\(show_title, which spent *r top_non_english\)total_weeks weeks** in the top 10.

Task 05: Longest Film in Global Top 10

{r longest-film} longest_film <- GLOBAL_TOP_10 %>% filter(!is.na(runtime)) %>% arrange(desc(runtime)) %>% slice(1) %>% mutate(runtime_minutes = round(runtime * 60))

The longest film to ever appear in the Netflix global Top 10 is r longest_film\(show_title* with a runtime of *r longest_film\)runtime_minutes minutes (r round(longest_film$runtime, 2) hours).

Task 06: Most Viewed Programs by Category

{r top-by-category} top_by_category <- GLOBAL_TOP_10 %>% group_by(category, show_title) %>% summarise(total_hours = sum(weekly_hours_viewed, na.rm = TRUE)) %>% ungroup() %>% group_by(category) %>% slice_max(total_hours, n = 1) %>% ungroup()

top_by_category %>% rename( Category = category, Show Title = show_title, Total Hours = total_hours ) %>% datatable(options = list(pageLength = 10), rownames = FALSE) %>% formatCurrency(‘Total Hours’, currency = ““, interval = 3, mark =”,“, digits = 0)

Task 07: Longest Country Top 10 Run for TV Show

{r longest-country-run} longest_run <- COUNTRY_TOP_10 %>% filter(category %in% c(“TV (English)”, “TV (Non-English)”)) %>% group_by(country_name, show_title, season_title) %>% summarise(weeks_in_top10 = max(cumulative_weeks_in_top_10, na.rm = TRUE)) %>% ungroup() %>% arrange(desc(weeks_in_top10)) %>% slice(1)

season_text <- ifelse(is.na(longest_run\(season_title), "N/A", longest_run\)season_title)

The TV show with the longest run in a country’s Top 10 is r longest_run\(show_title* (Season: r season_text) in *r longest_run\)country_name, which ran for r longest_run$weeks_in_top10 weeks**.

Task 08: Country with Limited Data

{r limited-data-country} country_weeks <- COUNTRY_TOP_10 %>% group_by(country_name) %>% summarise(total_weeks = n_distinct(week)) %>% ungroup() %>% arrange(total_weeks) %>% slice(1)

last_week <- COUNTRY_TOP_10 %>% filter(country_name == country_weeks$country_name) %>% summarise(last_date = max(week))

r country_weeks\(country_name* has limited data with only *r country_weeks\)total_weeks weeks of history. Netflix ceased operations in this country on r format(last_week$last_date, ‘%B %d, %Y’).

Task 09: Squid Game Total Viewership

{r squid-game} squid_game_total <- GLOBAL_TOP_10 %>% filter(str_detect(show_title, “Squid Game”)) %>% summarise(total_hours = sum(weekly_hours_viewed, na.rm = TRUE))

The total viewership of Squid Game across all seasons is r format(squid_game_total$total_hours, big.mark = “,”, scientific = FALSE) hours.

Task 10: Red Notice Views in 2021

{r red-notice} red_notice_2021 <- GLOBAL_TOP_10 %>% filter(show_title == “Red Notice”, year(week) == 2021) %>% mutate(estimated_views = round(weekly_hours_viewed / runtime)) %>% summarise(total_views = sum(estimated_views, na.rm = TRUE))

Red Notice received approximately r format(red_notice_2021$total_views, big.mark = “,”, scientific = FALSE) views in 2021.

Task 11: US Films That Climbed to #1

{r us-climbers} us_climbers <- COUNTRY_TOP_10 %>% filter(country_name == “United States”, category %in% c(“Films (English)”, “Films (Non-English)”)) %>% group_by(show_title) %>% mutate(hit_number_one = any(weekly_rank == 1), debut_rank = first(weekly_rank)) %>% filter(hit_number_one, debut_rank > 1) %>% ungroup() %>% arrange(desc(week)) %>% distinct(show_title, .keep_all = TRUE)

num_climbers <- nrow(us_climbers) most_recent_climber <- us_climbers %>% slice(1)

r num_climbers films reached #1 in the US after not debuting there. The most recent is r most_recent_climber$show_title.

Task 12: TV Show Charting in Most Countries (Debut Week)

{r most-countries-debut} debut_weeks <- COUNTRY_TOP_10 %>% filter(category %in% c(“TV (English)”, “TV (Non-English)”)) %>% group_by(country_name, show_title, season_title) %>% summarise(debut_week = min(week)) %>% ungroup()

most_countries <- COUNTRY_TOP_10 %>% filter(category %in% c(“TV (English)”, “TV (Non-English)”)) %>% inner_join(debut_weeks, by = c(“country_name”, “show_title”, “season_title”, “week” = “debut_week”)) %>% group_by(show_title, season_title) %>% summarise(num_countries = n_distinct(country_name)) %>% ungroup() %>% arrange(desc(num_countries)) %>% slice(1)

season_text2 <- ifelse(is.na(most_countries\(season_title), "N/A", most_countries\)season_title)

r most_countries\(show_title* (Season: r season_text2) charted in *r most_countries\)num_countries countries during its debut week.

Press Release 2: Netflix Success in India

india_growth <- COUNTRY_TOP_10 %>% filter(country_name == “India”) %>% mutate(year = year(week)) %>% group_by(year) %>% summarise(total_viewing_hours = sum(weekly_hours_viewed, na.rm = TRUE)) %>% ungroup() %>% arrange(year)

Netflix is experiencing remarkable success in India, the world’s most populous nation. non_english_reach <- COUNTRY_TOP_10 %>% filter(category %in% c(“Films (Non-English)”, “TV (Non-English)”)) %>% group_by(show_title) %>% summarise(countries = n_distinct(country_name)) %>% ungroup() %>% arrange(desc(countries)) %>% head(1)
